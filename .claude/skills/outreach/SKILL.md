# outreach — холодные письма, follow-up, КП

Фокус-скилл: один домен — тексты исходящих коммуникаций. Состояние клиента (статус,
история) живёт в `crm-pipeline`; сюда приходит уже выбранный лид + контекст.

## Когда брать / не брать

- ✅ «Напиши cold-письмо под этого клиента», «3 follow-up'а», «собери КП на лендинг»,
  «клиент говорит дорого — что ответить».
- ❌ «Добавь клиента / покажи воронку» → `crm-pipeline`. «Найди заказы» → `freelance-digest`.

## Процесс (плейбук)

1. **Контекст лида.** Возьми из брифа или `crm-pipeline`: кто, боль, роль, источник.
   Если нужно — быстрый ресёрч (deep-research) по компании/персоне.
2. **Квалификация BANT** (Budget / Authority / Need / Timeline) — коротко, чтобы не
   слать письмо нерелевантному адресату.
3. **Cold-письмо** по шаблону из `tools/templates.md`: **hook → ценность → CTA**,
   3-5 предложений. Первая строка — про КЛИЕНТА, не про себя. CTA конкретный
   («15 минут в четверг 11:00»), не «дайте знать».
4. **2-3 тональности** на выбор (экспертная / дружеская / прямая).
5. **Follow-up план**: день 3, день 7, день 14 — каждый под НОВЫМ углом (кейс,
   вопрос, «закрываю тему»). Шаблоны в `tools/templates.md`.
6. **КП** (если просят): структура из `tools/templates.md` — задача → решение →
   объём/сроки → цена → next step.
7. **Возражения**: ответы и углы — в `tools/objections.json`. Выясняй корневую
   причину, не дави.
8. **Отдай** тексты и сохрани в `agent-runtime/outputs/sales/<client-id>/`. Обнови
   клиента через `crm-pipeline` (статус/next_step).

## Правила

- Никаких штампов: «Надеюсь, у вас всё хорошо», «меня зовут… я представляю…».
- Первая строка цепляет и про клиента. Один CTA на письмо.
- Не обещай того, что не сделаешь. Не дави на возражениях — вскрывай причину.
- Длина cold-письма — короче лучше; КП — по делу, без воды.

## Слои этого скилла

- **tools/templates.md** — шаблоны cold-письма, follow-up #1/#2/#3, структура КП.
- **tools/objections.json** — справочник типовых b2b-возражений + углы ответа.
- **examples/cold-email-example.md** — few-shot эталон холодного письма.
