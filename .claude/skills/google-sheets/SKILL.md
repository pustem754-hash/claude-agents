# Google Sheets MCP Skill
This skill provides complete guidance for working with Google Sheets through the MCP integration. All tools have the prefix mcp__googlesheets__.
## Authentication
Before using any tool, authentication is required. Call mcp__googlesheets__authenticate first.
## Quick Reference
### Creating: create_google_sheet1, sheet_from_json
### Reading: batch_get, get_spreadsheet_info, get_sheet_names, lookup_spreadsheet_row, search_spreadsheets
### Writing: spreadsheets_values_append, batch_update
### Structure: add_sheet, delete_sheet, insert_dimension, delete_dimension
### Formatting: format_cell, set_basic_filter
### Charts: create_chart
### SQL: execute_sql, query_table
### Analytics: aggregate_column_data
