# freelance-digest — дайджест заказов с фриланс-бирж

Фокус-скилл: один проход «биржи → отфильтрованный дайджест». Outreach/CRM/КП сюда
не входят (Правило 3 — не монолит). Оркестрацию с `sales-agent` берёт на себя вызывающий.

## Когда брать / не брать

- ✅ «Собери свежие заказы», «фриланс-дайджест за сутки», «что на fl.ru по AI».
- ❌ «Напиши отклик на заказ», «добавь клиента в CRM», «сделай КП» → `sales-agent`.

## Процесс (плейбук)

1. **Параметры.** Возьми фильтры из аргументов пользователя; чего нет — из
   `tools/sources.json` → `default_filters` (свежесть 24ч, бюджет ≥1000р, темы).
2. **Сбор.** По каждой бирже из `tools/sources.json` → `exchanges` с рабочим
   `last_status` примени `method`. По умолчанию реально живой источник — **fl.ru**.
   Заблокированные (kwork 404, habr 410, workzilla JS) не долби — фиксируй статус.
3. **Фильтр.** Отбрось старше `freshness_hours` и дешевле `min_budget_rub`
   (бюджет «по договорённости» = `budget_rub: null`, НЕ отбрасывать — пропускать фильтр).
   Сопоставь тему из `topics`.
4. **Сложи JSON** по схеме из шапки `tools/build_digest_xlsx.py`. Путь:
   `agent-runtime/shared/freelance-raw-<YYYY-MM-DD>.json`.
5. **XLSX — кодом, не руками** (сквозной принцип). Запусти:
   ```
   python .claude/skills/freelance-digest/tools/build_digest_xlsx.py \
     agent-runtime/shared/freelance-raw-<DATE>.json \
     outputs/freelance-digest-<DATE>.xlsx
   ```
   Скрипт сам сортирует по бюджету, делает кликабельные ссылки и лист `meta`.
6. **Markdown-дайджест** по `tools/digest_template.md`. Эталон формата —
   `examples/digest-example.md`. Группируй по темам, внутри — бюджет desc, плюс
   блок «Топ-3 самых денежных».
7. **Отдай** оба пути (md + xlsx). Если канал Telegram — отправь xlsx как файл.

## Правила

- Бюджет распарси в число рублей (`budget_rub`); диапазон/этапы — суммируй, помечай в title.
- Никогда не выдумывай заявки. Нет данных с биржи — пиши статус «недоступен», не фантазируй.
- Дату ставь фактическую (today), не из примера.
- Скрипт `build_digest_xlsx.py` не переписывай — он детерминированный. Меняешь формат
  XLSX → правишь скрипт один раз, не генеришь заново.

## Слои этого скилла

- **tools/build_digest_xlsx.py** — детерминированная сборка XLSX (слой 3, код).
- **tools/sources.json** — справочник бирж + дефолт-фильтры (слой 3, конфиг).
- **tools/digest_template.md** — шаблон markdown-вывода.
- **examples/digest-example.md** — few-shot эталон (прогон 2026-04-16).
